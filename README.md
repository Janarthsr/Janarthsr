## About Me

I'm a second-year AIML student focused on **Physical AI** — the intersection of simulation, robot learning, and real-world deployment. My work spans humanoid control with MuJoCo, imitation learning via ACT (Action Chunking with Transformers), and sim-to-real pipelines targeting NVIDIA Jetson edge hardware.

Beyond robotics, I build applied AI systems — from phishing detection engines to AI-powered web apps for the Indian market.

Currently: Phase 2 of the **SO-101 Robotics Hackathon (XProbotics AI)**, building autonomous manipulation tasks with the LeRobot framework.

## Featured Projects

### 🤖 XP-Humanoid — Physical AI Framework
> MuJoCo · MediaPipe · YOLOv8 · Python · NVIDIA RTX 4050

End-to-end humanoid simulation with real-time human pose mirroring via MediaPipe, CUDA-accelerated YOLOv8 object detection, custom IMU-based fall prevention, and bipedal gait cycles. Designed for sim-to-real transfer to NVIDIA Jetson Orin Nano.

[![Repo](https://img.shields.io/badge/GitHub-Physical--AI-181717?style=flat-square&logo=github)](https://github.com/Janarthsr/Physical-AI)

### 🦾 SO-101 Robot Tasks — ACT Policy
> ACT · LeRobot · ROS 2 · MuJoCo · Docker · Python

Autonomous robot manipulation using Action Chunking with Transformers on the SO-101 robotic arm. Tasks include Pick & Place and Water Pouring, trained via imitation learning and deployed through the LeRobot framework.

[![Repo](https://img.shields.io/badge/GitHub-so101--robot--tasks-181717?style=flat-square&logo=github)](https://github.com/Janarthsr/so101-robot-tasks)

### 🛡️ PhishGuard — AI Cybersecurity System
> Scikit-learn · Google Gemini · Streamlit · Python-Telegram-Bot · SQLite

Dual-engine phishing detection combining ML classifiers (Random Forest/SVM) with Google Gemini for contextual threat analysis. Includes a Streamlit dashboard and real-time Telegram bot, tailored for the Indian digital landscape.

[![Repo](https://img.shields.io/badge/GitHub-PhishGuard-181717?style=flat-square&logo=github)](https://github.com/Janarthsr/PhishGuard)

### 🌾 AgriVeda — AI for Indian Farmers
> Gemini 1.5 Flash · JavaScript · HTML · Offline-first

Single-file web app for crop disease diagnosis via photo/camera, with offline fallback, multilingual support (English, Hindi, Kannada), live weather, and mock Mandi market prices for Karnataka. Built for Seedbrains 5.0 (200+ teams, UN SDG 2 alignment).
